# GNORD Windows Image SEO Documentation

This document records the existing GNORD website images that were reused in the static redesign. Each file was downloaded from the owned WordPress site, renamed with a descriptive SEO-friendly filename, and assigned human-readable alt text.

## What Changed

- Original generic filenames such as `1.png`, `2.png`, and `Untitled-design-3.png` were replaced with descriptive filenames.
- Empty alt attributes were replaced with specific image alt text.
- Important text that previously appeared only inside images was repeated as real HTML text, headings, captions, or body copy.
- Product-detail images are still used visually, while their product meaning is also represented in page text for search engines and AI answer engines.
